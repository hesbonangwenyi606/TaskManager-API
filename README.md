# Taskmanager API (Node.js + Express + MongoDB + JWT)

## Quick start

1. Copy `.env.example` to `.env` and set values.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server (dev):
   ```
   npm run dev
   ```
4. API base: http://localhost:3000/v1
5. Swagger UI: http://localhost:3000/v1/docs

## Features included
- Auth: register, login, refresh token (basic)
- Users: get profile, list (admin)
- Posts: CRUD with ownership checks
- MongoDB (Mongoose)
- JWT authentication
- Swagger docs (minimal)
